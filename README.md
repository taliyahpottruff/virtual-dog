# virtual-dog
A virtual dog bot for Discord.

## Environment Variables
This project uses dotenv to load environment variables. You can set them in any way that you want, but here is the .env template to get you started.
```
APPLICATION_ID=[Application ID]
TOKEN=[Token]
PUBLIC_KEY=[Public Key]
CLIENT_ID=[Client ID]
CLIENT_SECRET=[Client Secret]
PERMISSION_SCOPE=328565032960
```
